\Abstract{Palavras Chave}{Sistema conversacional de pesquisa, Descoberta de dados biomédicos, LLM, RAG, OMOP CDM, EHDEN, OHDSI.}
         {Resumo}{\selectlanguage{portuguese}A evidência do mundo real no domínio médico refere-se a evidências clínicas derivadas da análise de dados do mundo real. Isto tornou-se possível devido às estratégias atuais para lidar com os problemas de interoperabilidade de bases de dados. À medida que essas estratégias foram amplamente adotadas, novos desafios na descoberta de dados surgiram. Um desses desafios é a descoberta de bases de dados para estudos específicos de saúde. Inicialmente, isso foi tratado usando catálogos de bases de dados e interfaces para comparar características de bases de dados. No entanto, essas soluções tornaram-se menos eficazes à medida que as redes de bases de dados se expandem. Com o objetivo de melhorar a descoberta de bases de dados e simplificar a execução de estudos multicêntricos, esta dissertação propõe um sistema conversacional de pesquisa para ajudar os investigadores médicos a encontrar de forma eficiente as bases de dados mais adequadas para os seus estudos. A solução implementa a técnica BM25 para a recuperação ideal de bases de dados e utiliza um grande modelo de linguagem (LLM) para tarefas de geração de texto e processamento de linguagem natural (NLP). Os resultados demonstram que o sistema conversacional de pesquisa melhora significativamente a eficiência na descoberta de bases de dados, mas, mais importante, simplifica o processo de definição de coortes em bases de dados observacionais.}

\Abstract{Keywords}{Conversational query builder, Biomedical data discovery, LLM, RAG, OMOP CDM, EHDEN, OHDSI.}
         {Abstract}{Real-world evidence in the medical domain refers to clinical evidence derived from real-world data analysis. This become possible due to current strategies to handle the database interoperability issues. As these strategies have been widely adopted, new challenges in data discovery have appeared. One of these challenges is the discovery of databases for specific health studies. Initially, this was handled using database catalogs and interfaces to compare database characteristics. However, these solutions become less effective as database networks expand. Aiming to improve the discovery of databases and simplify the execution of multicenter studies, this dissertation proposes a conversational query builder to assist medical researchers in efficiently finding the most suitable databases for their studies. The solution implements the BM25 technique for optimal database retrieval and uses a large language model (LLM) for text generation and natural language processing (NLP) tasks. The results demonstrate that the conversational query builder significantly improves the efficiency of database discovery, but more importantly, it simplifies the process of defining cohorts over observational databases.}
